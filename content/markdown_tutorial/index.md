# Markdown教学

## 介绍

Markdown是一种轻量级标记语言，常用于生成文档、README文件和博客帖子。Markdown的特点是易读易写，可以使用文本编辑器快速生成具有格式的内容。

## 标题
Markdown中可以通过`#`符号来创建标题，标题的层级由`#`的数量决定：

| 级别 | Markdown语法| HTML语法|
|--|--|--|
| 一级  | `# 一级标题`     | `<h1>一级标题</h1>`   |
| 二级  | `## 二级标题`    | `<h2>二级标题</h2>`   |
| 三级  | `### 三级标题`   | `<h3>三级标题</h3>`   |
| 四级  | `#### 四级标题`   | `<h4>四级标题</h4>`   |
| 五级  | `##### 五级标题`   | `<h5>五级标题</h5>`   |
| 六级  | `###### 六级标题`   | `<h6>六级标题</h6>`   |

**注意：**
- 每个`#`后需跟一个空格。
- 不同级别的标题在视觉上存在层次感。

## 字体形式
可以通过以下方式改变字体样式：

|形式| Markdown语法| HTML语法|
|-|-|-|
|粗体|`**Bold**`|``<b>Bold</b>``|
|粗体|`__Bold__`|``<b>Bold</b>``|
|斜体|`*Italic*`|``<i>Italic</i>``|
|斜体|`_Italic_`|``<i>Italic</i>``|
|粗斜体|`***BI***`|``<b><i>BI</i></b>``|
|粗斜体|`___BI___`|``<b><i>BI</i></b>``|
|删除线|`~~Delete~~`|``<s>Delete</s>``|
|下划线|`<u>Underline</u>`|``<u>Underline</u>``|
|下标|`<sub>Sub</sub>`|``<sub>Sub</sub>``|
|上标|`<sup>Sup</sup>`|``<sup>Sup</sup>``|
|大字体|`<big>Big</big>`|``<big>Big</big>``|
|小字体|`<small>Small</small>`|``<small>Small</small>``|
|文本高亮|`<mark>TextHighlight</mark>`|``<mark>TextHighlight</mark>``|
|文本颜色|`<font color="red">文本</font>`|``<font color="red">文本</font>``|
|展示键盘输入|`<kbd>Ctrl</kbd>`|``<kbd>Ctrl</kbd>``|
|标记代码片段|`<code>代码示例</code>`|``<code>代码示例</code>``|
|表示注音或辅助文字|`<ruby>饕餮 <rt>tāo tiè</rt></ruby>`|``<ruby>饕餮 <rt>tāo tiè</rt></ruby>``|

#### 注意事项
- 1. 不同编辑器对`<mark>`标签的样式可能略有不同！
- 2. `<mark>`标签的样式可以用 `style="background-color: <颜色或#16进制颜色代码>;"` 来改变颜色
- 3. `<font>`标签的样式用 `color="<颜色或#16进制颜色代码>"` 来改变颜色

## 换行
换行的方式有三种：
1. **直接法**：在一句话后面加两个空格。
2. **空行法**：在两句话之间加一个空行。
3. **HTML 标签法**：使用 `<br>` 标签来换行。

## 空格
Markdown 中的空格可以通过以下 HTML 实体实现：

- `&nbsp;` : 不换行空格，默认宽度为四分之一em。
- `&ensp;` : 半角空格，宽度为半个em。
- `&emsp;` : 全角空格，宽度为一个em。
- `&thinsp;` : 窄空格，宽度为六分之一em。
- `&zwnj;` : 零宽不连字符。
- `&zwj;` : 零宽连字符。

## 插入Unicode字符
从 HTML 中插入 Unicode 字符的方式：

- **十进制**：以 `&#` 开头。
- **十六进制**：以 `&#x` 开头。

## 引用

使用 `>` 来创建引用，支持嵌套。示例如下：

``` md
> 这是引用
> > 这是引用的引用
> > > 这是引用的引用的引用
``` 
### 更多引用类型

#### 注意引用
``` md
> [!NOTE]  
> 强调用户在浏览时应考虑的信息。
``` 
#### 提示引用
``` md
> [!TIP]
> 帮助用户更好的可选信息。
``` 
#### 重要引用
``` md
> [!IMPORTANT]  
> 用户成功所需的重要信息。
``` 
#### 警告引用
``` md
> [!WARNING]  
> 因潜在风险，用户需要立即关注的重要内容。
``` 
#### 注意引用
``` md
> [!CAUTION]
> 操作可能带来的负面后果。
```

## 链接
创建链接的基本语法如下：

``` md
[链接名称](链接地址)
<链接地址>
``` 

### 增加链接 Title
可以为链接增加标题，使鼠标悬停时出现提示文字：

```md
[链接名称](链接地址 "markdown教程")
```
> **说明**：链接的 Title 是可选的，放在链接地址后面的引号内，用空格与链接地址分隔。

### 高级链接
使用变量创建链接，变量赋值在文档末尾：

``` md
这个链接用 1 作为网址变量 [Google][1]
这个链接用 runoob 作为网址变量 [Runoob][runoob]

然后在文档结尾为变量赋值（网址）：

[1]: http://www.google.com/
[runoob]: http://www.runoob.com/
``` 

## 图片
插入图片的基本语法如下：

```md
![图片描述文本](图片地址)
![图片描述文本](图片地址 "可选标题")
```
> **说明**：图片标题也是可选的。

## 列表
### 无序列表
使用 `*`、`+` 或 `-` 作为开头，后跟一个空格：

```md
* 这是个无序列表
+ 这是个无序列表
- 这是个无序列表
```

### 有序列表
使用数字加上 `.`，然后再加一个空格：

```md
1. 这是个有序列表
2. 这是个有序列表
```

### 嵌套列表
支持嵌套列表：

```md
* 这是个无序列表
   + 这是个无序列表内嵌的无序列表

- 这是个无序列表
   1. 这是个无序列表内嵌的有序列表

1. 这是个有序列表
   1. 这是个内嵌的有序列表
```


## 分割线
可以使用三个 `*` 或 `-` 创建分割线：

```md
***
---
```
> **注意**：
>
> 1. 建立分割线的前一行必须为空行。
> 2. 也可以使用 HTML 的 `<hr>` 标签：
>
> <hr> 

## 代码块
在Markdown中引用代码有两种方式：

### 1. 单行代码块
使用反引号（`）包围代码，以单行格式引用。示例：

``` md
`这是一个单行代码块`
``` 

### 2. 多行代码块
- **无高亮代码**：使用三个反引号（```）包裹代码。
  
  示例：
  ``` 
  这是一个多行代码块<br>
  仍然在这里
  ``` 

- **有高亮代码**：在开始的三个反引号后，添加语言名称进行高亮。

  示例（高亮Python代码）：
  ``` python
  def hello():
      print("Hello, world!")
  ``` 


## 表格
表格使用`|`来分割单元格，使用`-`来分隔表头与其它行。格式说明：

- `:-`：表头左对齐
- `:-:`：表头居中
- `-:`：表头右对齐

### 表格示例：
``` md
| 1 | 2 | 3 | 4 |
| --- | :----- | :-----: | ----: |
| 正常 | 居左 | 居中 | 居右 |
```

| 1 | 2 | 3 | 4 |
| --- | :----- | :-----: | ----: |
| 正常 | 居左 | 居中 | 居右 |

## 脚注
脚注的语法如下所示：

``` md
脚注一号标记这里[^1]
[^1]: 这是脚注一号的解释
``` 

> 注意：
> 1. 脚注会自动移动到文章的后面。

## 待办事项
使用方括号来创建待办事项列表：

``` md
- [ ] 未完成
- [x] 已完成
``` 

## 书写公式
使用KaTeX来书写数学公式。示例：

``` md
单行公式：
$3 + 3 = 6$

多行公式：
$$
2 + 2 = 4
$$
``` 

## 展开折叠

代码如下([Losketch](http://github.com/Losketch)提供):

使用`<details>`标签来创建可展开的折叠部分。格式如下：

``` markdown
<details>
<summary>标题</summary>
<!--这里要一行空行，否则里面的Markdown样式无法生效-->

</details>
```
